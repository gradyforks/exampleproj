Installation

  • Step 1 : Extract BEEMEDIA.zip file from Files folder to server.
  • Step 2: Install Composer Dependencies - You will need to install the composer dependencies in your application by running the following command inside the root directory of your app. Install Composer
  • Step 3 : Create database using BEEMEDIA.sql from Database folder.
  • Step 4 : Edit details of .env file in project folder in server.
  • Step 5 : make sure you storage symlink is created by running the following command : 
    php artisan storage:link
